מערכת לניהול טיול שנתי

המערכת בנויה על מנת לתת מענה להורים ולמרוים על מיקום התלמידים בכל זמן נתון

שפות וטנולוגיות:
**שפות**
צד שרת - API
צד ךקוח - react
דטה - SQLite

**טכנולוגיות**
צד שרת - node.js, express
צד לקוח - leaflet

**הנחות מקלות**
*הורה יכול לגשת למיקום בן על פי ת"ז
* מורה יכול לראות מיקום תלמיים
* מורה יכול להוסיף תלמיד
* מנהל יכול לראות תלמידים ומרוים
* מנהל יכול להוסיף תלמידים ומורים

  **התקנות:**
לפתוח טרמינל חדש
·         cd backend
·         npm install
·         npm run dev

לפתוח טרמינל חדש
·         cd frontend
·         npm install
·         npm run dev

**שימוש במערכת**
בדף הבית יש אפשרות לכניסה לדף מציאת מיקום תלמיד על ידי תעודת זהות
וכן דף התחברות למורה
לאחר התחברות יעברו לדף מורה שם תוצג מפה עם מיקום התלמידות ורשימת התלמידות שרחוקות מהמורה
בנוסף מתאפשר למורה להוסיף תלמיד

לאחר התחברות מנהל יעבור לדף בו יראה את מיקום כל התלמידים וכן יוכל להוסיף כיתה ומורה
